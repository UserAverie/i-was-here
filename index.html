This website is a place for students to experiment with working in git. You can submit a pull request to this project.

It is expected that all students will make a change to the code. The change should be small and should follow these rules.

  1. You should make a change to the CSS styles for the page.
  2. You should leave your student number somewhere in the visible HTML.
  3. You should not change the HTML in this section.
  4. You should not remove another student number from the code.

Obviously if you take significant effort to update the CSS rules, another student may follow this up with another change that destroys your impact. This is just how it is. So don't get too precious about it. Just try to make an incremental improvement.
